###   `sudo pip3 install 6-rosdep`

###      通过此就可以将rosdep的包通过国内来进行安装了。速度就会比较迅速了

### 5.`rosrun rqt-robot-streeing rqt-robot-streeing`### 可以使用这个工具对仿真机器人进行模拟
